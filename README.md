# LiveKit AI Auctioneer

An interactive voice-based auction system powered by LiveKit and AI. This application creates an engaging auction experience where participants can place bids while an AI auctioneer manages the proceedings in real-time.

## Features

- **AI Auctioneer**: Powered by Llama 3.1 70B model, providing dynamic and contextual responses
- **Voice Interaction**: Real-time speech-to-text and text-to-speech capabilities
- **Live Bid Management**: Tracks and announces new highest bids
- **Multi-participant Support**: Handles multiple bidders in real-time
- **Interruption Handling**: Smart interruption system to ensure clear bid announcements

## Prerequisites

- Python 3.12+
- LiveKit server access
- Required API keys:
  - LiveKit API key and secret
  - Cerebras API key
  - OpenAI API key (optional, if using OpenAI instead of Cerebras)
